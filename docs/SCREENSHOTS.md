# Screenshots Guide

Use this guide to prepare polished portfolio screenshots.

## Capture Checklist
- Landing page hero + architecture sections
- Dashboard with live cards/charts
- Alerts panel expanded details
- AI log upload and analysis output
- Mobile responsive dashboard view

## Naming Convention
- `dashboard-overview.png`
- `ai-log-analysis.png`
- `infrastructure-health.png`
- `landing-architecture.png`
- `mobile-dashboard.png`

## Recommended Size
- Desktop: 1920x1080
- Mobile: 1080x1920

## Placement
Store final files in `screenshots/` and reference them from `README.md`.
