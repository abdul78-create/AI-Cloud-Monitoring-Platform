# Setup Guide

## Frontend
1. `cd frontend`
2. `npm install`
3. Copy `.env.example` to `.env.local`
4. `npm run dev`

## Backend
1. `cd backend`
2. `npm install`
3. Copy `.env.example` to `.env`
4. `npm run dev`

## Ollama
1. Install Ollama from [https://ollama.com](https://ollama.com)
2. Pull model: `ollama pull llama3`
3. Start Ollama service locally
